## 介绍

一个轻量级的自定义浏览器导航页面，具有以下特点：

- 🔍 支持多搜索引擎快速切换

- 📑 可自定义书签分组，支持拖拽排序

- 💾 使用本地存储保存用户配置

- 📱 自适应桌面端和移动端

- 🔄 支持导入导出配置

- ☁️ 集成 WebDAV 备份功能

- 🎨 简洁现代的界面设计

## 技术栈

- 原生 JavaScript（无框架）
- HTML5 & CSS3
- Local Storage API

## 本地开发

1. 克隆仓库后直接在浏览器中打开 index.html 即可
2. 修改代码后刷新页面即可看到效果
3. 建议使用 Live Server 等工具提供实时预览
